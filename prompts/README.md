# Prompt Library (Java • Spring Boot • Microservices)

Use these prompts with Cursor/Claude Code (tasks), VS Code (snippets), and IntelliJ (Live Templates). See `conventions/CONVENTIONS.md` for naming rules and prefixes.

## Quick Start

- **VS Code**: Import `vscode/java-prompts.code-snippets` via *User Snippets* or project `.vscode/`
- **IntelliJ**: Import `intellij/java-prompts-live-templates.xml` via *Settings → Editor → Live Templates*
- **Cursor/Claude Code**: Import `cursor/tasks.json` into your tasks/playbooks

## Privacy

Redact secrets/tokens. Prefer enterprise/on‑prem models if policy requires.
